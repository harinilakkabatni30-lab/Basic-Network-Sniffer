 Basic Network Sniffer
 CodeAlpha Cyber Security Internship - Task 1
 Objective
 Develop a Python-based Network Sniffer to capture and analyze network packets.
 Tools Used
- Python 3
- Scapy
  Features
- Captures network packets
- Displays Source IP Address
- Displays Destination IP Address
- Displays Protocol Number
- Displays Packet Length
How to Run
1. Install Scapy:
   pip install scapy
2. Run:
   python packet_sniffer.py
Learning Outcomes
- Packet capturing
- Network traffic analysis
- IP addressing
- Network protocols
